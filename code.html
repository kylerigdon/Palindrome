The Code

                    
                        function getValues(){

                            let inputString = document.getElementById('inputString').value;
                            inputLower = inputString.toLowerCase().replace(/[\W_]/g, '');
                          
                            if (inputLower == 0) {
                              Swal.fire({
                                icon: "error",
                                title: "Oops!",
                                text: 'Please enter text.',
                                backdrop: false
                              });
                            } else {
                          
                              let reversedString = [];
                          
                            for (let i = inputLower.length - 1; i >= 0; i--){
                              reversedString += inputLower[i];
                            }
                              let checkedPalindrome = checkForPalindrome(inputLower, reversedString);
                          
                              displayResults(checkedPalindrome, reversedString);
                            }
                          }
                          
                          function checkForPalindrome(inputLower, reversedString){
                          
                            let palindrome = ''
                          
                            if (reversedString == inputLower){
                              palindrome = 'true'
                            } else {
                              palindrome = 'false'
                            }
                          
                            return palindrome;
                          }
                          
                          function displayResults(checked, reversed){
                          
                            if (checked == 'true'){
                              Swal.fire({
                                icon: "success",
                                title: "Well done! You entered a Palindrome!",
                                text: `Your text reversed is: ${reversed}`,
                                backdrop: false
                              });
                            } else {
                              Swal.fire({
                                icon: "error",
                                title: "Sorry. You did not enter a Palindrome.",
                                text: `Your text reversed is: ${reversed}`,
                                backdrop: false
                               });
                            }
                          
                          }

The code is structured in three functions.

getValues()

The getValues function collects the input from the user using the value of the html element with the id of inputString and creates a variable inputString with that value. Then using .toLowerCase() and .replace(/[\W_]/g, '') it will make all letters lower case and remove any special characters or spaces and pass the mutated value over to the new variable inputLower. Once thats finished it will run inputLower through an if/else statement to make sure the user has entered something in, if they have not it will display a message prompting them to please input text. After it has verified that the user has input text it will head to the decrementing for loop which will reverse the letters in the string and add them to populate the array called reversedString. Then the reversedString and inputLower are passed to the checkForPalindrome function and that function will return a value that will be stored in checkedPalindrome. Finally checkedPalindrome and reversedString's values are passed to the displayResults function.

checkForPalindrome()

The checkForPalindrome function takes the values of inputLower and reversedString, creates an empty variable to use later. Then it will run an if/else statement to check if the inputLower value and the reversedString's values are the same, if they are the same it will pass a value of true to the variable called palindrome if they are not the same it will pass the value of false. Once those conditions are checked it will return the variable of palindrome's value back to the getValues function for it to use.

displayResults()

The displayResults function will take the values of the checkedPalindrome variable and the reversedString variable, then it will execute a if/else statement to check if the checked value is true or false. If it is true it will display a message using Sweetalert2 that says "Well done! You entered a Palindrome!" and then in the lower half of the message show the phrase or word in reverse. If it is false it will display a message saying "Sorry. You did not enter a Palindrome." and show the word in reverse.
